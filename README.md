# SQL Journey

A personal repository documenting my SQL learning path — from fundamentals to advanced query writing.
Problems are self-designed with custom datasets, organized by topic and difficulty.

---

## About

Each `.sql` file contains:
- **Database setup** — CREATE TABLE + INSERT statements (commented out after first run)
- **Problems** — Clearly labeled Q001, Q002... with difficulty tags
- **Solutions** — Written directly below each problem as commented queries
